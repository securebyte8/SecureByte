Steps to execute the Project
1.Download and extract the "SecureByte 1.0 zip" file
2.Open the main folder
3.Double click the “SecureByte1.0.exe” application
4.Choose any one of the tools from the 2 modules
5.You can give the input as per the tool
6.The output will be display on the window
7.You can view the full report of the tools in the third module
8.You can also create the shortcut for the application to run from desktop
